# CodeAlpha Calculator

A modern and responsive Calculator built using HTML, CSS, and JavaScript as part of the Frontend Development Internship at CodeAlpha.

## 🚀 Features

- Basic arithmetic operations
  - Addition (+)
  - Subtraction (−)
  - Multiplication (×)
  - Division (÷)
- Clear screen functionality
- Delete last character
- Real-time result calculation
- Keyboard support
- Responsive and modern UI
- Hover effects and stylish design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ⌨️ Keyboard Support

| Key | Function |
|-----|----------|
| 0-9 | Numbers |
| + - * / | Operators |
| Enter | Calculate Result |
| Backspace | Delete Last Character |
| C | Clear Screen |
| . | Decimal Point |
